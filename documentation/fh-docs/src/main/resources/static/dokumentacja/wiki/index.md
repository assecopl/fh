## Linki 

Linki do plików *.md obsługiwane są wewnatrz kontrolki.
Linki zewnętrzne otwierane są zawsze w nowej karcie.

```
- [Wewnętrzny relatywny link](wew/dokumentacja.md)

- [Zewnętrzny link](http://google.pl)
```
